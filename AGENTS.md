# Cerberus

Cerberus是一个new-api的fork，其是我们的商用API管理平台，需要为我们的业务场景做出定制。

## 主要场景

我们主要对外销售各类模型的API，下游可以进行下一级分发，也可以自行接入使用。

## 说明

我们的Claude模型由../cc-bridge提供，如果功能需要，可能需要改动其代码。

## 约束

- 保持良好的架构，代码改动需要长远考虑，不要只知道最小化改动（除非是线上问题）
- 对于业务请求（模型API请求）内容的修改，如果涉及到闭源模型（claude、gpt等），必须谨慎考虑是否会造成关联影响，防止封号
- 对于充值、支付、消费等功能，必须反复考虑并自检，防止出现bug或漏洞
